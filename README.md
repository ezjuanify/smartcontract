# Smart Contract

## Description
This project demonstrates implementation of Smart Contract with Spring Boot.

## Endpoints
### /send
- Parameter 1: **from** account id (string)
- Parameter 2: **to** account id (string)
- Parameter 3: **amount** (double)
### Response
- return code
- id

### /get-balance
- Parameter 1: **account** id
### Response
- return code
- amount
- id
